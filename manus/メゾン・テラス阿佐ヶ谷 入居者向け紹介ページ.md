# メゾン・テラス阿佐ヶ谷 入居者向け紹介ページ

## 概要
このプロジェクトは、メゾン・テラス阿佐ヶ谷の入居者向け紹介ページです。
温かみと高級感を兼ね備えたデザインで、ファミリー層に好まれるウェブサイトとなっています。

## ファイル構成
- `introduction.html` - メインの紹介ページ
- `style.css` - スタイルシート
- `images/` - 物件の画像ファイル（JPG形式）
- `material.md` - 物件情報の元データ

## デザインの特徴
- **配色**: ベージュ/アイボリー系の温かみのある背景色
- **アクセントカラー**: ゴールド（#d4af37）で高級感を演出
- **フォント**: Noto Serif JP（明朝体）で上品な印象
- **レイアウト**: ゆったりとした余白で読みやすさを重視

## 使用方法
1. introduction.htmlをブラウザで開く
2. すべての画像が正しく表示されることを確認
3. GitHubリポジトリにアップロード

## GitHubへのアップロード手順
```bash
# ファイルをステージング
git add introduction.html style.css images/*.jpg

# コミット
git commit -m "Add introduction page"

# プッシュ（認証が必要）
git push origin main
```

## 注意事項
- 画像ファイルはHEIC形式からJPG形式に変換済み
- ブラウザ互換性のため、Web標準に準拠したコードを使用
